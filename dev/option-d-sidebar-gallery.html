Galley
What is this?

A lightweight editor for shared HTML documents. You edit text in the browser; changes write directly back to the file.

How it works
  1. Click a document to open it
  2. Click any text to edit — look for the blue outline on hover
  3. Use the toolbar for bold, italic, and links
  4. Press Ctrl+S or click Save to write changes to disk
Good to know

Every save creates a backup. If someone else edits the file while you're working, you'll be notified. Press ? in the editor for keyboard shortcuts.
